# StackOverflow Developer Survey (2024) Language Analysis

This project analyzes data from the 2024 StackOverFlow Developer Survey to identify meaningful trends across digital industries in relation to the developer expierence (Job, Language Favorites, Industry, etc).


Potential Clients: Technology Ed platforms or general schools that would like to more closely align content towards the trends of real developers, Startup companies that want to create tools for the next gen of developers, so their product will be more popular/useful

Questions Asked
- Q1: Which languages do developers most want to learn? What language has the most equal desire-admire rate
- Q2: Are certain industries strongly associated with specific programming languages
- Q3: Which programming languages are associated with the highest salaries among survey respondents?
- Q4: Which languages have the strongest overlap with high job satisfaction?

Dataset Source Link: https://survey.stackoverflow.co/
